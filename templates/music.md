# 음악 노트 템플릿

## 4차원 해석 (음악)

| 차원 | 음악에서의 의미 |
|------|---------------|
| **craft** | 사운드·프로덕션·편곡·연주·믹싱·보컬 기술 |
| **narrative** | 가사·앨범 흐름·곡 간 연결·스토리 (앨범 단위) |
| **impact** | 정서적 울림·몸의 반응·잔상 |
| **rewatch** | 재청 가치 (반복 청취해도 새로운가) |

## Frontmatter

```yaml
---

type: music

year:

created: YYYY-MM-DD

revisited_at: 
status: completed | in-progress | abandoned
rating: 0.0
craft: 0
narrative: 0
impact: 0
rewatch: 0
tags:
  - music
---
```

> **단위 가이드**:
> - 앨범 단위로 듣는 경우 → `unit: album`, `title`은 앨범명
> - 곡이 하나만 박혔으면 → `unit: single`, `title`은 곡명, `album`도 채움

## Phase 1 질문 셋 (음악)

### 공통

```
1. 어떤 작품인지 한 단락 (장르·결·소속)
2. 다 듣고 가장 먼저 떠오른 감정 한 단어 + 왜
3. 가장 인상적인 순간 (특정 곡? 한 구절? 한 사운드?)
4. 좋았던 점
5. 아쉬웠던 점
```

### 매체 특화 — 음악

```
A1. 사운드의 결이 어땠어? (밀도·공간·톤·텍스처)
A2. 보컬에 대한 인상 — 음색? 발성? 감정 전달?
A3. 가사 중에 박힌 라인 있어? (옮겨와줘)
A4. 편곡·악기 중에 두드러진 게 있었어? (베이스? 드럼? 신스? 스트링?)
A5. 앨범이면 — 곡 순서·흐름이 의도적으로 느껴졌어?
A6. 듣는 *상황*이 떠올라? (혼자? 출퇴근? 새벽? 작업 중?)
A7. 본인이 음악 만드는 사람이라면, 너의 음악 작업에 영향을 준 부분이 있어?
```

> A7은 사용자가 음악 창작자(보컬리스트, 작곡가, 프로듀서 등)일 때 활성화하는 매체 특화 질문.
> 다른 매체에 없는 "내가 만드는 사람으로서 본 것"을 끌어내는 자리.

## 작성 노트

- 아티스트·앨범은 옵시디언 위키링크: `[[The Weeknd]]`
- 같은 아티스트의 다른 앨범 노트가 있으면 `## 함께 듣기`로 연결
- 단일곡 노트가 여러 개면 나중에 앨범 노트에서 위키링크로 모음
- A7 (음악 만드는 사람의 시선)은 선택적 — 적용되는 작품에서만 활성화
