## Phase 1 질문 셋 (영화)

### 공통

```
1. 줄거리 한 단락 (스포일러 OK)
2. 다 보고 가장 먼저 떠오른 감정 한 단어 + 왜
3. 가장 인상적인 장면 (한 컷, 한 시퀀스, 한 대사)
4. 좋았던 점
5. 아쉬웠던 점
```

### 매체 특화 — 영화

```
M1. 시각적으로 가장 강했던 장면 (구도? 색? 빛? 카메라 움직임?)
M2. 사운드(음악·앰비언스·정적) 중에 박힌 게 있어?
M3. 편집의 결 — 빠르게 몰아붙였어, 길게 머물게 했어?
M4. 캐릭터 중에 가장 *입체적*이었던 인물? 왜?
M5. 메타포 / 상징 / 모티프 중에 떠오르는 거 있어? (없으면 OK)
M6. 감독의 *서명*이 보였어? 어떤 부분에서?
M7. 보면서 / 보고 나서 *질문*이 생긴 게 있어?
```

## 작성 노트

- 감독·배우는 옵시디언 위키링크: `[[봉준호]]`, `[[송강호]]`
