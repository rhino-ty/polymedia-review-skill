# 게임 노트 템플릿

## 매체 변형 (4D → 5D)

> SKILL.md의 기본 4D + 10점 척도에서 다음 변형 적용. 다른 매체 template은 변형 없음.

| 변형 | 내용 | 이유 |
|------|------|------|
| **차원 추가** | `systems` 축 신규 (총 5D) | 게임은 *상호작용 매체*. 메커닉·게임플레이 루프·디자인 일관성은 책·영화·음악엔 없는 게임 고유의 평가축. `craft`에 우겨넣으면 *시청각 완성도*와 *재미의 본질*이 한 점수에 뭉개진다. |
| **명칭 변경** | `rewatch` → `replay` | 의미(재경험 가치)는 동일하나 게임 매체에 맞는 표현. 게임의 재플레이는 *설계 의도*(빌드 다양성·다회차 변주·하드 모드 등)까지 포함하므로 의미 강화. |
| **차원 재정의** | `craft` = 시청각·기술 완성도 (게임플레이는 `systems`로 분리) | 4D 시절 craft가 *모든 비-서사 요소*를 떠안고 있었음. systems 분리 후 craft를 시청각·폴리시·UI 중심으로 좁힘. |
| **척도 변경** | 10점 정수+소수 → **5점 + 소수 한 자리** (예: `4.2/5`) | 게임은 장르 폭이 매우 넓어 *과도한 granularity*가 점수 흔들림을 유발. 5점 소수(1.0~5.0, 50단계)는 충분히 정밀하면서 표기 간결. |
| **N/A 룰** | `narrative` N/A 허용 (비서사 게임) → 해당 4축만 평균 | 로그라이크·캐주얼·FPS·격투 등 절반 가까운 장르가 서사 없음. 강제 점수화 X. 예: craft 3.0 / systems 4.0 / narrative `-` / impact 3.0 / replay 4.0 → (3+4+3+4)/4 = **3.5/5**. |

## 5차원 해석 (게임)

| 차원 | 게임에서의 의미 |
|------|---------------|
| **craft** | 시청각·기술 완성도 (아트·사운드·애니메이션·폴리시·UI/UX) |
| **systems** | 메커닉·게임플레이 루프·디자인 일관성·게임 필 *(게임 전용 축)* |
| **narrative** | 스토리·세계관·캐릭터·연출 *(비서사 게임은 `-` 표기)* |
| **impact** | 몰입·체험·시간이 흐른 감각·정서적 울림 |
| **replay** | 재플레이 가치 (다른 빌드·엔딩·하드 모드 시도 의지 포함) |

## Phase 1 질문 셋 (게임)

### 공통

```
1. 작품의 골격 한 단락 (스토리 + 핵심 게임플레이)
2. 다 깨고 / 멈추고 가장 먼저 떠오른 감정 한 단어 + 왜
3. 가장 인상적인 순간 (스토리 장면? 보스전? 선택지? 시스템 깨달음?)
4. 좋았던 점 — 떠오르는 대로
5. 아쉬웠던 점 — 있으면
```

### 매체 특화 — 게임 (5D 대응)

```
G1. 조작감은 어땠어? (단단함, 무게감, 반응성, 손맛)
    → systems 핵심 + craft 보조

G2. 시청각(그래픽·사운드·UI)이 *몰입*에 얼마나 기여했어?
    → craft 단독 측정. systems와 분리 인터뷰 필수.

G3. 시스템(전투/성장/경제 등) 중에 *체화*된 감각이 있었어?
    → systems 깊이

G4. 처음 ~30분과 끝 ~30분 중 어느 쪽이 더 인상적이었어?
    → impact + replay

G5. 게임이 너에게 "선택"을 강제한 순간이 있었어? 어떤 선택?
    → narrative + impact

G6. 멈췄거나 다 깬 후 — 캐릭터 / 세계관 / 시스템 중 가장 오래 남는 건?
    → 5축 분포 진단

G7. 플레이타임 vs 만족도 — 시간 가성비? (긴데 못 멈춘 / 짧은데 깊은 / 길어서 늘어진)
    → replay 직결
```

> **인터뷰 주의**: craft와 systems는 게임에서 *섞이기 쉬움*. G1·G2·G3를 분리해서 묻고, 답변도 분리해서 받기. 사용자가 craft(시각·사운드) 얘기 중인지 systems(메커닉·게임 필) 얘기 중인지 매번 분류.

## 작성 노트

- 개발사·플랫폼은 옵시디언 위키링크: `[[Supergiant Games]]`
- Roguelike vs Roguelite는 정확히 구분 (메타 성장 유무)
- 미완료(abandoned) 상태도 노트 작성 가능 — 왜 멈췄는지가 중요한 데이터
- **craft ↔ systems 경계 가이드**: "조작감"은 *systems* (반응성·게임 필), "UI/HUD 디자인"은 *craft* (시각 완성도). 헷갈리면 G1·G2를 분리해서 다시 인터뷰.
- **점수 척도 정합**: 게임 노트는 5점 소수. 다른 매체 노트는 10점. 매체끼리 비교 시 `rating`을 정규화(게임 점수 × 2 = 10점 환산).
