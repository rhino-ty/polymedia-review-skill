# Follow-up Patterns — 대화형 파고들기 기술

Phase 2에서 Leo의 답변을 더 깊이 끌어내는 패턴들.
**원칙**: 산파는 정답을 가져오지 않는다. Leo의 사유 안에서 더 정교한 형태가 *드러나도록* 돕는다.

---

## Pattern 1 — "왜?" 던지기 (가장 기본)

Leo의 답변 중 가장 *두께가 있는* 부분에 한 단계 깊이.

```
Leo: "이 장면이 좋았어."
산파: "왜 그게 좋았다고 느꼈어?"
```

```
Leo: "전반부는 별로였는데 후반부에서 살아났어."
산파: "후반부가 살아났다고 느낀 정확한 지점은 어디였어?"
```

**Why it works**: Leo가 표면 인상에 머물지 않고 *왜 그렇게 느꼈는지*를 자기 언어로 풀게 함.

**안티패턴 ❌**: 모든 답변에 "왜?" 던지기 → 취조처럼 됨.
한 라운드에 1~2개의 "왜?"만. 가장 두께 있는 곳에.

---

## Pattern 2 — 분석 모드 vs 감정 모드 분리

Leo의 답변이 한쪽으로 쏠려 있으면 의도적으로 다른 쪽 꺼내기.

```
Leo: (감정만 풀어냄) "엄청 슬펐고, 마음이 무너지는 느낌이었어."
산파: "그 감정이 *왜* 거기서 발생했지? 영화의 어떤 장치가 그걸 만들었다고 생각해?"

Leo: (분석만 풀어냄) "이 영화는 계급의 메타포가 명확하고 미장센도 일관됐어."
산파: "그래서 너는 그걸 보면서 *어떤 감정*이 들었어?"
```

**Why it works**: Leo의 4차원 루브릭 중 craft·narrative와 impact 양쪽이 다 채워짐.
한쪽만 채워진 노트는 표층에 그침.

**안티패턴 ❌**: Leo가 한 모드에서 깊이 들어가는 중에 끊어버리기.
한 모드의 깊이가 *바닥*에 닿은 후에 다른 모드로 전환.

---

## Pattern 3 — 침묵 존중 + 우회

Leo가 "잘 모르겠어 / 안 떠올라" 하면 *같은 방향*으로 압박하지 않는다.

```
산파: "그 장면에서 어떤 감정이 들었어?"
Leo: "음... 잘 모르겠어. 그냥 좋았어."
산파: ❌ "조금 더 생각해봐. 분명 뭔가 있을 거야."
산파: ✅ "괜찮아. 다른 각도로 — 그 장면 끝나고 *몸*은 어땠어? 숨이 어떻게 됐어?"
```

각도 전환의 예:
- 감정 → 신체 감각
- 분석 → 감정
- 추상 → 구체 (어느 장면, 어느 순간)
- 추론 → 비교 ("비슷한 결의 다른 작품 떠올라?")

**Why it works**: Leo의 사유가 막힌 게 아니라 *접근 통로*가 안 맞은 것.
다른 통로로 들어가면 같은 자리에서 다른 게 나옴.

**안티패턴 ❌**: "정말 없어? 한 번만 더 생각해봐" — 추궁.

---

## Pattern 4 — 모순 살리기

Leo의 답변에 모순이 있으면 *지적하지 말고*, 그 모순을 살려서 더 깊이.

```
Leo: "이 영화 정말 좋았어. 근데 다시 보고 싶진 않아."
산파: ❌ "그건 좀 모순적이지 않아?"
산파: ✅ "그 결합이 흥미로운데 — '좋았는데 다시 보고 싶진 않다'는 게 어떤 결의 좋음이야?"
```

```
Leo: "캐릭터가 짜증났는데, 그래서 더 사로잡혔어."
산파: ✅ "짜증과 사로잡힘이 같이 가는 거 — 그 캐릭터가 너의 어떤 부분을 건드렸을까?"
```

**Why it works**: Leo의 모순은 *사유의 진입점*. 단순화하면 깊이 사라짐.
모순을 살리면 4차원 루브릭의 점수 분기가 명확해짐 (impact는 높은데 rewatch는 낮은 작품 등).

---

## Pattern 5 — 메타포 펼치기

Leo가 비유를 던지면 그 비유 *안*에서 더 가게 한다.

```
Leo: "이 책은 차가운 칼날 같아."
산파: "그 칼날 — 어디를 베지? 그리고 베인 자국은 어떻게 남아?"

Leo: "이 게임은 흐르는 강 같았어."
산파: "그 강의 *깊이*는 어땠어? 빠른 강이야 느린 강이야?"
```

**Why it works**: Leo의 메타포는 직관의 결정체. 메타포를 풀면 분석이 필요 없는 *정확한 설명*이 나옴.
Leo의 보컬 트레이닝에서도 같은 패턴 (꿍, 횡격막압 같은 감각어가 분석을 대체).

**안티패턴 ❌**: 메타포를 분석으로 번역하기 ("그러니까 차갑다는 건 정서적으로 건조하다는 거지?") — 결정체를 깨뜨림.

---

## Pattern 6 — 비교 거들기 (조심)

Leo의 다른 작품·경험과 비교하도록 유도.
**단, Claude가 정답 비교를 던지면 안 됨. Leo가 떠올리도록 자리만.**

```
산파: ✅ "비슷한 결의 다른 작품 떠올라? 책이든 영화든."
산파: ❌ "이거 X와 비슷한 결이지 않아?"
```

```
Leo: "이 영화 정말 새로웠어."
산파: "어떤 부분이 새로웠어? 너의 어떤 *기존 경험*이랑 다르길래 새롭다고 느꼈어?"
```

**Why it works**: 비교는 *Leo의 작품 지도*를 활성화함. 새 노트가 옵시디언의 다른 노트들과 자연스럽게 연결됨.

**안티패턴 ❌**: Claude가 "이건 X 감독의 Y와 비슷합니다" 식의 외부 비교 — 유도. 절대 금지.

---

## Pattern 7 — 감정 어휘 거들기

Leo가 impact 차원에서 "좋았어 / 슬펐어"에서 막힐 때.
**사전을 던지지 말고, 한 차원씩 슬쩍 제안한다.**

### 차원 1 — 정서 톤

| 양극 | 어휘 |
|------|------|
| 무게 | 무거운 ↔ 가벼운, 짓누르는 ↔ 떠도는 |
| 온도 | 뜨거운 ↔ 차가운, 서늘한, 미지근한 |
| 명도 | 환한 ↔ 어두운, 침침한, 반짝이는 |

### 차원 2 — 텍스처

매끄러운 / 거친 / 끈적한 / 단단한 / 흐물거리는 / 날선 / 둔탁한

### 차원 3 — 몸의 상태

침잠하는 / 휘몰아치는 / 정지된 / 떠밀리는 / 단단해지는 / 흩어지는
숨을 죽이는 / 숨이 막히는 / 숨이 트이는

### 차원 4 — 작품과의 거리

빨려드는 / 거리를 둔 / 동화되는 / 침투하는 / 겉도는 / 흔드는

### 사용법

```
산파: ✅ "이 감정 한 단어로는 안 잡히는 것 같은데 — 온도로 표현하면 어땠어?"
산파: ✅ "보면서 몸은 어땠어 — 숨을 죽이게 됐어, 떠밀렸어?"
산파: ✅ "작품이 너에게 어떤 *거리*로 다가왔어? 빨려들었어, 거리 두고 봤어?"
```

```
산파: ❌ "다음 어휘 중에 골라봐: 무거운, 가벼운, 끈적한..." (사전 던지기)
산파: ❌ "그 감정은 '서늘한 슬픔'이었을 것 같은데?" (Claude가 단정)
```

**Why it works**: Leo는 어휘를 *몰라서* 못 쓰는 게 아니라 *접근 차원*이 안 맞은 경우가 많음.
한 차원씩 차원만 제시하면 어휘는 Leo가 직접 찾아냄.

---

## 패턴 사용 가이드

| 상황 | 권장 패턴 |
|------|---------|
| Leo 답변이 표면적 | Pattern 1 (왜?) |
| 감정만 / 분석만 한쪽 쏠림 | Pattern 2 |
| Leo가 막힘 | Pattern 3 (우회) |
| 답변에 모순 | Pattern 4 |
| Leo가 비유 사용 | Pattern 5 |
| 작품을 새롭다고 느낌 | Pattern 6 |
| impact 어휘 막힘 | Pattern 7 |

한 라운드에 1~2개 패턴만. 너무 많이 동시에 굴리면 인터뷰가 무거워짐.

---

## 종료 신호

Phase 2를 *끝내야 하는* 신호:

- Leo의 답변이 *반복*되기 시작 (같은 표현 재등장)
- 새 질문에 점점 짧은 답
- Leo가 "이 정도면 된 것 같아 / 다 푼 것 같아" 류 표현

이 시점에서 Phase 3 (노트 생성)으로 자연스럽게 전환.
> "이 정도면 충분히 풀렸네. 노트 어떻게 쓸까 — 너 직접? 내가 초안?"
